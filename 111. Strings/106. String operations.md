# String operations

## Problem Description
Akash likes playing with strings. One day he thought of applying following operations on the string in the given order:

Concatenate the string with itself. </br>
Delete all the uppercase letters. </br>
Replace each vowel with '#'. </br>
You are given a string A of size N consisting of lowercase and uppercase alphabets. Return the resultant string after applying the above operations.

***NOTE:*** 'a' , 'e' , 'i' , 'o' , 'u', are defined as vowels.

### Problem Constraints
1<=N<=100000

---

## Input Format
First argument is a string A of size N.

## Output Format
Return the resultant string.

---

## Example Input
- **Input 1:** </br>
A="aeiOUz"

- **Input 2:** </br>
A="AbcaZeoB"

## Example Output
- **Output 1:**
"###z###z"

- **Output 2:** </br>
"bc###bc###"

## Example Explanation
- **Explanatino 1:** </br>
First concatenate the string with itself so string A becomes "aeiOUzaeiOUz". </br>
Delete all the uppercase letters so string A becomes "aeizaeiz". </br>
Now replace vowel with '#', A becomes "###z###z".

## Explanatino 2: </br>
First concatenate the string with itself so string A becomes "AbcaZeoBAbcaZeoB". </br>
Delete all the uppercase letters so string A becomes "bcaeobcaeo". </br>
Now replace vowel with '#', A becomes "bc###bc###".

---

# CODE

```python
class Solution:
    # @param A : string
    # @return a strings
    def string_operation(self, A):
        a = A + A
        B = []

        for i in range(len(a)):
            if a[i].islower():
                B.append(a[i])

        for i in range(len(B)):
            if B[i] in ['a', 'e', 'i', 'o', 'u']:
                B[i] = '#'

        return ''.join(B)


solution = Solution()
print(solution.string_operation(A="AbcaZeoB"))  -->  O/P: "bc###bc###"
```
