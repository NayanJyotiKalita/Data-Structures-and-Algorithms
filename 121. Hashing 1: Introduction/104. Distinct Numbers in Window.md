# Distinct Numbers in Window

## Problem Description
You are given an array of N integers, A1, A2 ,..., AN and an integer B. Return the of count of distinct numbers in all windows of size B. </br>
Formally, return an array of size N-B+1 where i'th element in this array contains number of distinct elements in sequence Ai, Ai+1 ,..., Ai+B-1. </br>

***NOTE:*** if B > N, return an empty array. </br>


### Problem Constraints
1 <= N <= 106 </br>
1 <= A[i] <= 109

---

### Input Format
First argument is an integer array A </br>
Second argument is an integer B.

### Output Format
Return an integer array.

---

## Example Input
```
Input 1:
 A = [1, 2, 1, 3, 4, 3]
 B = 3
 
Input 2:
 A = [1, 1, 2, 2]
 B = 1
```

## Example Output
```
Output 1:
 [2, 3, 3, 2]
 
Output 2:
 [1, 1, 1, 1]
```

Example Explanation
```
Explanation 1:
 A=[1, 2, 1, 3, 4, 3] and B = 3
 All windows of size B are
 [1, 2, 1]
 [2, 1, 3]
 [1, 3, 4]
 [3, 4, 3]
 So, we return an array [2, 3, 3, 2].
 
Explanation 2:
 Window size is 1, so the output array is [1, 1, 1, 1].
```

---

# CODE

```pthon	
class Solution:
	# @param A : list of integers
	# @param B : integer
	# @return a list of integers
	def dNums(self, A, B):
        hm = {}

        #EdgeCase if B is greater than size of A, then return empty
        if B > len(A):
            return []

        result = []

        #count distinct elemtn in window size B
        for i in range(B):
            if A[i] in hm:
                hm[A[i]] += 1
            else:
                hm[A[i]] = 1
        
        result.append(len(hm))  #append the elements count to the result
        
        #slide the window    
        l = 0
        r = B
        while r < len(A):
            hm[A[l]] -= 1       #decrement the left element count

            if hm[A[l]] == 0:   #if left element count is zero, remove A[L] from hashmap
                del hm[A[l]]
            
            if A[r] in hm:      #incremetn or add the count  right element
                hm[A[r]] += 1
            else:
                hm[A[r]] = 1
            
            result.append(len(hm))

            l += 1
            r += 1
        
        return result


solution = Solution()
print(solution.dNums( A = [1, 2, 1, 3, 4, 3], B = 3 ))  -->  O/P:  [2, 3, 3, 2]
```
