# Find Time Complexity - 1

## Q. What is the time complexity of the following code snippet

```C++
for(int i = 1 ; i <= n ; i+=2){
    count << i ;
}
```

```Java
for (int i = 1; i <= n; i += 2) {
    System.out.print(i);
}
```

```Python
for i in range(1, n + 1, 2):
    print(i, end='')
```

### Options:
    1. O(n ^ 3)
    2. O(n ^ 2)
    3. O(n)
    4. O(logn)

---

## Answer
3
