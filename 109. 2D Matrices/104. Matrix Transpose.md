# Matrix Transpose

## Problem Description
Given a 2D integer array A, return the transpose of A. </br>
The transpose of a matrix is the matrix flipped over its main diagonal, switching the matrix's row and column indices.

### Problem Constraints 
1 <= A.size() <= 1000  </br>
1 <= A[i].size() <= 1000 </br>
1 <= A[i][j] <= 1000

---

### Input Format
First argument is a 2D matrix of integers.

### Output Format
You have to return the Transpose of this 2D matrix.

---

## Example Input
- Input 1: </br>
A = [[1, 2, 3],[4, 5, 6],[7, 8, 9]]

- Input 2: </br>
A = [[1, 2],[1, 2],[1, 2]]

## Example Output
- Output 1: </br>
[[1, 4, 7], [2, 5, 8], [3, 6, 9]]

- Output 2: </br>
[[1, 1, 1], [2, 2, 2]]

## Example Explanation
- Explanation 1: </br>
Clearly after converting rows to column and columns to rows of [[1, 2, 3],[4, 5, 6],[7, 8, 9]] </br>
 we will get [[1, 4, 7], [2, 5, 8], [3, 6, 9]].

- Explanation 2: </br>
After transposing the matrix, A becomes [[1, 1, 1], [2, 2, 2]]

---

CODE:

```python
# for n x m matrix where n != m
class Solution:
    # @param A : list of list of integers
    # @return a list of list of integers
    def matrix_transpose(self, A):
        c = []
        for j in range(len(A[0])):
            inner_list = []
            for i in range(len(A)):
                inner_list.append(A[i][j])
            c.append(inner_list)
        return c
```

'or'

```python
# for n x n matrix
class Solution:
    # @param A : list of list of integers
    # @return a list of list of integers
    def matrix_transpose(self, A):
        n = len(A)
        for i in range(n):
            for j in range(i+1, n):
                A[i][j], A[j][i] = A[j][i], A[i][j]
        return A
```

```python
solution = Solution()
print(solution.matrix_transpose(A = [[1, 2, 3],[4, 5, 6],[7, 8, 9]]))  -->  O/P: [[1, 4, 7], [2, 5, 8], [3, 6, 9]]
```
